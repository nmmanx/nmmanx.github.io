# X2Vocab

Hi! I'm building **X2Vocab**, a mobile app that helps you learn vocabulary *twice as fast* with the help of generative AI.

## The problems it solves

When learning a new language, vocabulary is the first thing you need to master. X2Vocab uses *dynamic flashcards* to help you build a vocabulary list and learn it in the most effective way.

You may find it useful if you’ve faced these common issues with flashcards:

- Learning from a pre-built deck doesn’t match your level or give you words useful for your life.
- Building your own vocabulary flashcards takes too much time (translations, examples, genders, base forms…).
- Flashcards often lack contextual examples — even if you add one, it’s static, so you only learn the word in one context.
- Adding images to flashcards is a hassle.

X2Vocab does all of this automatically for you.  
It uses LLMs to generate **dynamic flashcards** so you can learn in multiple contexts quickly.  
It also integrates an image search tool, so you don’t have to manually find, download, and attach pictures yourself.  
Plus, it has a built-in spaced repetition algorithm to help you remember words as efficiently as possible.

## How am I building it?

- **Frontend:** Kotlin Multiplatform  
- **Backend:** Appwrite  
- **AI:** Gemini Flash Lite 2.0  
- **Image search:** Powered by Pexels

I am currently targeting Android, but I will consider supporting iOS soon when I have a MacBook :\(\(.
